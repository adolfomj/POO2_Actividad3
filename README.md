# POO2_Actividad3
# Sistema de Gestión de Empleados

Este proyecto consiste en una aplicación web desarrollada con Spring Boot que permite la gestión de empleados mediante operaciones CRUD (Crear, Leer, Actualizar y Eliminar).

---

## Tecnologías utilizadas

- Java
- Spring Boot
- Maven
- Thymeleaf
- JPA (Hibernate)
- Base de datos H2

---

## Arquitectura

El proyecto está organizado bajo el patrón de diseño **MVC (Modelo - Vista - Controlador)**:

- **Model**: Representa la entidad Employee (EmployeeEntity)
- **Repository**: Maneja el acceso a datos con CrudRepository
- **Service**: Contiene la lógica de negocio y validaciones
- **Controller**: Gestiona las peticiones HTTP
- **View**: Interfaces HTML con Thymeleaf

---

## Funcionalidades

- ✅ Listar empleados  
- ✅ Agregar empleados  
- ✅ Eliminar empleados  
- ✅ Validación de datos  
- ✅ Ordenamiento con programación funcional (Streams)

---

## Conceptos aplicados

- Programación Orientada a Objetos (POO)
- Patrón MVC
- Patrón Repository
- Patrón Service
- Programación funcional (Streams y Lambdas)
- Validaciones y manejo de errores
